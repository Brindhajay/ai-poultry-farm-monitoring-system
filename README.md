# AI Poultry Farm Monitoring and Temperature Regulation System

## Overview

This project is an IoT-based smart poultry farm monitoring system that automatically monitors and regulates environmental conditions such as temperature, humidity, and air quality using ESP8266 and sensors.

It helps improve poultry health, reduce manual work, and prevent losses.

---

## Features

• Real-time temperature monitoring  
• Humidity monitoring  
• Air quality monitoring  
• Automatic temperature regulation  
• Firebase cloud integration  
• LCD live display  
• IoT remote monitoring  

---

## Hardware Used

• ESP8266 NodeMCU  
• DHT22 Temperature and Humidity Sensor  
• MQ135 Gas Sensor  
• LCD Display  
• Cooling Fan  
• Heater  

---

## Software Used

• Arduino IDE  
• Firebase Realtime Database  
• Embedded C  
• IoT
